# Emojipack Generator

Generate Alfred Emoji packs.

## Usage

Build Emoji Pack and open with Alfred. Be careful to uncheck the "Strip
snippets of 'auto expand' flag" checkbox in the confirmation dialog that
appears.

```sh
just install
```
